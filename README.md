# png2snes  
Conversor moderno e completo de PNG → SNES, ideal para romhacks e homebrew.

Permite converter gráficos de qualquer resolução e organizá-los adequadamente em:
- Tiles SNES (`.chr`)
- Tilemaps SNES (`.map`)
- Paletas SNES (`.pal`)
- Paletas GIMP (`.gpl`)
- Metatiles opcionais (`.meta.json`)
- Pré-visualização de tiles (`-tileset.png`)
- Merge automático de múltiplas partes (`final.chr/.map/.pal`)

Compatível com BG Mode 1 e BG Mode 3 (BG1).  
Sprites também suportados.
